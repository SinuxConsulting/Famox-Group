# Famox Group (Demo)

This is a client-demo build for the Famox Group website redesign.

## Tech
- React 18 + Vite + TypeScript
- React Router
- Tailwind CSS + shadcn/ui

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Admin
- URL: `/admin`
- Demo password: `famoxadmin123`

> Demo data is stored locally (browser localStorage). No database/email integrations are enabled in this demo build.
